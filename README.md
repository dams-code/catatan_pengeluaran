# Sistem Catatan Pengeluaran Damar

Aplikasi web sederhana berbasis Django untuk mencatat pengeluaran harian.

## Fitur

- Menambah data pengeluaran
- Melihat daftar pengeluaran
- Mengubah data pengeluaran
- Menghapus data pengeluaran
- Mengelola kategori pengeluaran

## Teknologi

- Python
- Django
- SQLite

## Cara Menjalankan Project

```bash
python manage.py runserver
